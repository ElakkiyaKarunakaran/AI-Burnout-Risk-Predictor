**AI Burnout Risk Predictor**
🚀 Live Project: https://ai-burnout-risk-prediction.vercel.app/
**Project Overview**

The AI Burnout Risk Predictor is a machine learning-based web application designed to identify the risk of burnout in individuals based on their daily habits and lifestyle patterns. Burnout is a growing issue among students and working professionals due to excessive workload, stress, and poor work-life balance.

This project uses artificial intelligence to analyze user inputs such as work hours, sleep duration, screen time, stress level, and physical activity to predict whether a person is at Low, Medium, or High risk of burnout.

**Problem Statement**

Many individuals experience burnout without realizing the early warning signs. Organizations and individuals often lack simple tools to monitor mental fatigue and workload imbalance.

This project aims to provide an early detection system that can help users understand their burnout risk and take preventive actions.

**Key Features**

Predicts burnout risk using a machine learning model

Accepts user inputs related to daily lifestyle and work habits

Provides instant prediction results

Simple and user-friendly web interface

Helps users understand their mental health and workload balance

**Machine Learning Model**

The burnout prediction model was trained using supervised machine learning techniques. The dataset was preprocessed, cleaned, and split into training and testing sets. The model was then trained to learn relationships between lifestyle factors and burnout risk levels.

To evaluate model performance, techniques such as:

Residual Analysis

Prediction Accuracy

Model Evaluation Metrics

were used.

**How It Works**

The user enters lifestyle details in the web form.

The inputs are sent to the Flask backend.

The trained machine learning model processes the inputs.

The system predicts the burnout risk level.

The result is displayed instantly on the webpage.
